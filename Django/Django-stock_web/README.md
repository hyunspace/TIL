## 프로젝트 시작

```bash
$ python -m venv venv
$ source venv/Scripts/activate

$ pip install django==3.2.12

$ django-admin startproject {프로젝트명} .
```



앱 추가

```bash
$ python manage.py startapp {앱명}
```
